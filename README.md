# Common Ground Ecological Planning

Static marketing website for an ecological planning practice focused on turning field observations and geospatial data into practical stewardship, restoration, monitoring, and funding decisions.

## Local preview

From this directory, run:

```powershell
python -m http.server 8000 --bind 127.0.0.1
```

Then open `http://127.0.0.1:8000/`.

## Hosting

The production site is deployed from the root of the `main` branch with GitHub Pages.
